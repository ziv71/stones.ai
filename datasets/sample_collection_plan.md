# Sample collection plan

## Goal
Collect enough labeled images to train a practical stone classifier for the app.

## Recommended workflow
1. Start with one class at a time.
2. Collect 50-100 clean images per class first.
3. Review them for duplicates and low-quality photos.
4. Move a portion into validation and test folders.
5. Re-train and compare results.

## Suggested class list
- granite
- basalt
- marble
- sandstone
- limestone
- slate

## Capture guidelines
- Shoot close to the stone surface.
- Keep the stone centered.
- Use consistent framing where possible.
- Capture both texture and full-object views.
- Include some varied lighting conditions.
